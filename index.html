Introduction

Python is a high-level, interpreted programming language known for its simplicity and readability. Python's syntax allows developers to express concepts in fewer lines of code than in other languages like C++ or Java.

Key features of Python:

  • Easy to learn and use
  • Readable and maintainable code
  • Extensive standard library
  • Dynamic typing and automatic memory management
  • Cross-platform compatibility
Getting Started
Installation

Python can be downloaded from the official website python.org. It is available for Windows, macOS, and Linux.

To install Python:

  1. Download the latest version of Python for your operating system.
  2. Run the installer and follow the instructions.
  3. Ensure that Python is added to your system's PATH environment variable (this is an option during installation).
Hello, World!

Once Python is installed, you can start programming! Open a terminal or command prompt and type:

 print("Hello, World!")

This will output:

 Hello, World!
Basic Syntax
Variables and Data Types

Python supports several data types, including:

  • Integers (int)
  • Floating-point numbers (float)
  • Strings (str)
  • Booleans (bool)

Example:


  x = 10             # Integer
  y = 3.14           # Float
  name = "Python"    # String
  is_active = True   # Boolean
Operators
  a = 10
  b = 5

  # Arithmetic operators
  print(a + b)  # 15
  print(a - b)  # 5
  print(a * b)  # 50
  print(a / b)  # 2.0

  # Comparison operators
  print(a == b)  # False
  print(a > b)   # True
Conditionals

Conditionals are used to execute code based on certain conditions. Python uses if, elif, and else.


  x = 20
  if x > 10:
      print("x is greater than 10")
  elif x == 10:
      print("x is equal to 10")
  else:
      print("x is less than 10")
Loops

Python supports both for and while loops.

For Loop:


  for i in range(5):
    print(i)

While Loop:


  i = 0
  while i < 5:
      print(i)
      i += 1
Functions

  def greet(name):
    return f"Hello, {name}!"

  print(greet("Alice"))

Functions can accept parameters and return values:


  def add(x, y):
    return x + y

  result = add(3, 4)
  print(result)  # 7
Modules and Packages

Python has a rich standard library, and you can import external modules using the import statement.


  import math

  print(math.sqrt(16))  # 4.0
Creating Modules

You can also create your own modules by saving Python code in a .py file.

Example:

  1. Create a file mymodule.py:
    2. 
  def greet(name):
    return f"Hello, {name}!"
  2. Import mymodule in another Python file:
    3. 
  import mymodule

  print(mymodule.greet("Bob"))
Object-Oriented Programming

Python supports object-oriented programming (OOP). The basic concepts include:

  • Classes
  • Objects
  • Methods
  • Inheritance
Creating a Class

  class Dog:
    def __init__(self, name, breed):
        self.name = name
        self.breed = breed
    
    def bark(self):
        print(f"{self.name} is barking!")

  # Create an object
  dog1 = Dog("Fido", "Labrador")
  dog1.bark()  # Output: Fido is barking!
Error Handling

Python uses try, except, and finally blocks to handle errors.


  try:
      x = 10 / 0
  except ZeroDivisionError:
      print("Cannot divide by zero!")
  finally:
      print("Execution completed.")
Libraries and Frameworks

Python has many popular libraries and frameworks for different purposes, including:

  • Data Science: numpy, pandas, matplotlib, scikit-learn
  • Web Development: Flask, Django, FastAPI
  • Machine Learning: TensorFlow, PyTorch
  • Automation: selenium, requests
Best Practices
  1. Use meaningful variable names to improve readability.
  2. Follow PEP 8, the official style guide for Python code.
  3. Write comments and documentation to explain your code.
  4. Use list comprehensions and other Pythonic idioms to make your code more concise.
  5. Avoid global variables—use functions and classes to encapsulate state.
